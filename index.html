Turntable.fm Bot

Sparkle Turntable Bot is the most popular free room bot on Turntable.fm. This bot is written in Node.JS and uses an optional MySQL server for room statistics. This project is developed and maintained by @sharedferret.

Features

This bot can:

  • Manage a waitlist/queue for a room
  • Enforce room rules regarding song limits and timeouts before a person can DJ again
  • Respond to a set of commands in chat and through TT.FM's new PM system
  • Report song stats after each song
  • Welcome users to the room via chat and PM
  • Receive and respond to commands via HTTP requests
  • Log events to the console and/or a MySQL database

Get your own

This guide will help you set up a copy of the bot in your Turntable.fm room. While it should be relatively straightforward, it does assume that you have a basic understanding of using a shell or command line. Since Sparkle Turntable Bot is designed to run 24 hours a day, installation on a server is optimal, but the bot will run fine on a personal computer.
